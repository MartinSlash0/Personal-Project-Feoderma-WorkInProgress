Hi there! My name is Martin and this is my personal project.

After several months of learning in the Frontend Developer Career Path, I've decided to realise one of my ideas which is this solo project. This is a site for my fiance's cosmetics and massage studio.

How I stay committed to learning

I like to think of myself as a lifelong learner. I used to spend hours and hours learning, then try to create simple projects using what I learned or work new techniques into existing projects.

How I got started

I started couple of months ago when I got tired of my last job as a physiotherapist. I've always had interest in programming but never got into it, after a talk with some of my friends that work in this field I decided to give it a try and here I am now. I've enrolled in a Front End Developer Career Path course at Scrimba in Fabruary this year.

The meaning of this project

With this project im trying to show you all the skills I've acquired through the duration of this course.
